# Linux Client/Daemon - Restart

**Restart the daemon or bird2**

## Options

| Shorthand | Option   | Parameter                | Description | Default |
| --------- | -------- | ------------------------ | ----------- | ------- |
|           | --daemon | restart this client      |             | [X]     |
|           | --bird2  | restart the bird2 daemon |             |         |

## Usage

```bash
user@bgp-router:~$ sudo rk-client restart
```
